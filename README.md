### 🎬 [BlockPost](https://github.com/tung-programming/BlockPost)
**Decentralized Media Ownership & Verification Platform**<br>
![React](https://img.shields.io/badge/React-blue?logo=react) ![Node.js](https://img.shields.io/badge/Node.js-green?logo=nodedotjs) ![Polygon](https://img.shields.io/badge/Polygon-9000ff?logo=polygon) ![IPFS](https://img.shields.io/badge/IPFS-65C2CB?logo=ipfs)

BlockPost is a decentralized content platform where creators upload media, compute multi-layer perceptual + audio hashes, pin content to IPFS via dual providers, and optionally register ownership proofs on-chain. Built for transparency, copyright evidence, and duplicate detection across a trustless network.

- 🔍 Multi-layer duplicate detection: SHA-256, perceptual hash & audio fingerprint
- 📦 Dual IPFS pinning (Pinata + Web3.Storage) for redundancy
- ⛓️ On-chain ownership & VideoGuard verification (Polygon)
- 🦊 MetaMask-integrated creator profiles & feed
- 🚀 [Live Demo →](https://block-post.vercel.app/)

**Real-Time Aerial Object Detection System**<br>
![Python](https://img.shields.io/badge/Python-3670A0?logo=python&logoColor=ffdd54) ![FastAPI](https://img.shields.io/badge/FastAPI-lightgrey?logo=fastapi) ![React](https://img.shields.io/badge/React-blue?logo=react) ![PyTorch](https://img.shields.io/badge/PyTorch-red?logo=pytorch)

SkyGuard is a full-stack drone detection system powered by a custom-trained YOLOv8 model. It supports images, video uploads, live webcam feeds, and mobile IP camera streams — all through a React web interface backed by a FastAPI inference server with GPU acceleration.

- 🎥 Multi-source input: image, video, webcam, mobile (RTSP/HTTP)
- ⚡ CUDA-accelerated FP16 inference on custom `best.pt` model
- 🔔 Pushbullet real-time alerts on drone detection
- 📐 Rotation control & dynamic tab switching
- 🧪 CLI inference tool + full dataset preparation pipeline

### ⚡ [GridSense AI — Smart Electricity Forecasting](https://github.com/tung-programming/4th-SEM-MAIN_EL)
**ML-Driven Energy Prediction & Grid Analytics Engine**<br>
![Python](https://img.shields.io/badge/Python-3670A0?logo=python&logoColor=ffdd54) ![scikit-learn](https://img.shields.io/badge/scikit--learn-%23F7931E?logo=scikit-learn) ![XGBoost](https://img.shields.io/badge/XGBoost-EC4B27?logo=python) ![React](https://img.shields.io/badge/React-blue?logo=react)
